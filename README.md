se quiere implementar la funcion de listar en "Terminal" utilizando las clases base de listar en las vistas y conectandolas con los controladores.

Equipaje:
por cada registro debe mostrar:
el id del equipaje
el peso del equipaje con todos sus decimales (sin formato exponencial)
el nombre del duenio del equipaje
el nombre publico de la imagen

Pasajero:
por cada registro debe mostrar:
la cedula del pasajero
el nombre del pasajero
la edad del pasajero
genero del conductor (si es true se debe mostar como masculino y si es false como femenino)
el nombre publico de la imagen

Tiquete:
el id del tiquete
el nombre del pasajero del tiquete
el viaje de esta manera (ciudadOrigen-ciudadDestino)
el id del asiento
el estado del tiquete (si es true mostrar "ocupado" de color verde, si es false mostrar "desocupado" de color rojo)
el nombre publico de la imagen

Viaje:
el id del viaje
la ruta del viaje de esta manera (ciudadOrigen-ciudadDestino)
el nombre del conductor del bus
el id del bus
la fecha del viaje
la hora del viaje
el estado del viaje (si es true mostrar "en curso" de color verde, si es false mostrar "finalizado" de color rojo)

se quiere implementar la funcion de listar en "Terminal" utilizando las clases base de listar en las vistas y conectandolas con los controladores.

Equipaje:
por cada registro debe mostrar:
el id del equipaje
el peso del equipaje con todos sus decimales (sin formato exponencial)
el nombre del duenio del equipaje
el nombre publico de la imagen

Pasajero:
por cada registro debe mostrar:
la cedula del pasajero
el nombre del pasajero
la edad del pasajero
genero del conductor (si es true se debe mostar como masculino y si es false como femenino)
el nombre publico de la imagen

Tiquete:
el id del tiquete
el nombre del pasajero del tiquete
el viaje de esta manera (ciudadOrigen-ciudadDestino)
el id del asiento
el estado del tiquete (si es true mostrar "activo" de color verde, si es false mostrar "inactivo" de color rojo)
el nombre publico de la imagen

Viaje:
el id del viaje
la ruta del viaje de esta manera (ciudadOrigen-ciudadDestino)
el nombre del conductor del bus
el id del bus
la fecha del viaje
la hora del viaje
el estado del viaje (si es true mostrar "en curso" de color verde, si es false mostrar "finalizado" de color rojo)
el nombre publico de la imagen

Asiento:
el id del asiento
el id del bus
el estado del asiento (si es true mostrar "ocupado" de color verde, si es false mostrar "desocuopado" de color rojo)
el nombre publico de la imagen

Ruta:
el id de la ruta
la ciudad de origen
la ciudad de destino
la tarifa de la ruta con todos sus decimales (sin formato exponencial)
el nombre publico de la imagen

Conductor:
la cedula del conductor
el nombre del conductor
la edad del conductor
el genero del conductor (si es true se debe mostar como masculino y si es false como femenino)
el nombre publico de la imagen

Bus:
el id del bus
el modelo del bus
el nombre de la empresa del bus
el nombre publico de la imagen

Empresa:
el id de la empresa
el nombre de la empresa
el nombre publico de la imagen

para hacer estas implementaciones, trabaja sobre las clases que ya han sido creadas, no borres ni edites codigo a menos que sea estrictamente necesario para su funcionamiento.
